# MediConnect - Doctor Appointment System

A modern full-stack web application for booking doctor appointments easily.

## Features
- Search doctors by specialty.
- View doctor profiles and availability.
- Book and manage appointments.
- User-friendly UI with Tailwind CSS.

## Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS
- **State Management:** React Context API
- **Icons:** SVG icons for specialties

## Getting Started
1. Clone the repository: `git clone https://github.com/kuldeeps01199/MediConnect.git`
2. Install dependencies: `npm install`
3. Run the project: `npm run dev`
